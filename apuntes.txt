git reset HEAD --deshace los cambios del area de staging area del ultimo commit(si hacemos un git add.)

git revert HEAD --deshace el ultimo cambio

git fech --busca los cambios

git diff --staged o git diff --cached: Muestra las diferencias entre los archivos en el área de preparación y el último commit confirmado.
git diff commit1 commit2: Muestra las diferencias entre dos commits específicos. Puedes especificar los commits usando sus hashes, nombres de ramas, o referencias de etiquetas.
git diff branch1 branch2: Muestra las diferencias entre dos ramas diferentes.
git diff etiqueta1 etiqueta2: Muestra las diferencias entre dos etiquetas diferentes.
